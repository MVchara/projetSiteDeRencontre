Pour que le php et le sql marchent, il faut se connecter à phpmyadmin
 et importer config.sql.
Il faut aussi dans le fichier config.php remplir $username et $password
 avec respectivement le nom d'utilisateur et le mot de passe de connexion à phpmyadmin.


Pour se connecter en tant qu'admin, il faut aller sur la page
Espace Admin depuis l'accueil et entrer :
			utilisateur: admin
			mdp: admin1234
	
	
Pour devenir abonné, il faut ,dans la page pour s'abonner, entrer comme code abonné : lhoub2024
